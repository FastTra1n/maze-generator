# MazeGenerator 🧩

**MazeGenerator** — это консольное приложение для генерации лабиринтов и поиска кратчайшего пути в нём.

## Функционал

- Настройка размера лабиринта (ширина/высота);
- Выбор алгоритма генерации лабиринта (DFS, Kruskal);
- Выбор алгоритма поиска пути (A*, BFS);
- Возможность выбора начальной и конечной точек;
- Просмотр результата.

## Зависимости

- Компилятор с поддержкой C++17;
- CMake 3.10+.

## Сборка проекта

Для сборки проекта выполните следующие команды из корня проекта:
```bash
cmake -S . -B build;
cmake --build build;
cd build;
./maze_generator
```
